Privacy Policy

Last updated: May 2026 · Applicable to: SOP Skill Chrome Extension

In short: SOP Skill does not collect, store, or transmit any of your personal data to our servers. All processing happens locally in your browser.

1. Data We Do NOT Collect

SOP Skill does not have any backend servers. We do not collect:

  • Personal identification information (name, email, etc.)
  • Browsing history or website content
  • Recorded workflows or SOP documents
  • Screenshots or audio recordings
  • Usage analytics or telemetry
  • Cookies or tracking identifiers

2. Data Stored Locally

All data is stored exclusively in your browser using chrome.storage.local:

  • API keys — Credentials for speech-to-text providers (Deepgram or Aliyun) that you configure yourself
  • Recording sessions — Captured actions, screenshots, and voice narrations during a recording
  • User preferences — Language and UI settings

This data never leaves your device and is automatically deleted when you uninstall the extension or clear your browser data.

3. Third-Party Services

SOP Skill optionally connects to the following third-party services for speech-to-text functionality. These features are entirely optional and only activate when you provide your own API key:

Service Purpose Data Sent
Deepgram Real-time speech recognition Audio from your microphone (real-time stream)
Alibaba Cloud (Aliyun Qwen) Real-time speech recognition Audio from your microphone (real-time stream)

Audio data is streamed in real-time via WebSocket and is not stored by SOP Skill. Please review the privacy policies of these providers:

  • Deepgram Privacy Policy
  • Alibaba Cloud Privacy Policy

4. Screenshots

SOP Skill captures screenshots of web pages during recording sessions to document your workflow. These screenshots are:

  • Processed entirely within your browser
  • Annotated locally using Canvas API in an offscreen document
  • Never uploaded to any server
  • Embedded only in locally generated SOP documents

5. Permissions Explained

SOP Skill requests only the permissions necessary for its core functionality:

  • activeTab — Access the current tab when you start recording
  • sidePanel — Display the recording UI in Chrome's side panel
  • storage — Save your settings and API keys locally
  • scripting — Inject the action capture script into web pages
  • clipboardWrite — Copy SOP text to clipboard when you click "Copy"
  • offscreen — Process screenshots for click-position annotation
  • webNavigation — Detect page navigations during recording

6. Children's Privacy

SOP Skill is not directed at children under 13 and does not knowingly collect any information from children.

7. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date.

8. Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.

隐私政策

最后更新：2026年5月 · 适用于：SOP Skill Chrome 扩展

简而言之：SOP Skill 不会收集、存储或向我们的服务器传输您的任何个人数据。所有处理均在您的浏览器中本地完成。

1. 我们不收集的数据

SOP Skill 没有任何后端服务器。我们不收集：

  • 个人身份信息（姓名、邮箱等）
  • 浏览历史或网页内容
  • 录制的工作流或 SOP 文档
  • 截图或录音
  • 使用分析或遥测数据
  • Cookie 或追踪标识符

2. 本地存储的数据

所有数据仅通过 chrome.storage.local 存储在您的浏览器中：

  • API 密钥 — 您自行配置的语音转文字服务提供商（Deepgram 或阿里云）凭证
  • 录制会话 — 录制过程中捕获的操作、截图和语音讲解
  • 用户偏好 — 语言和界面设置

这些数据永远不会离开您的设备，卸载扩展或清除浏览器数据时会自动删除。

3. 第三方服务

SOP Skill 可选连接以下第三方服务以提供语音转文字功能。这些功能完全可选，仅在您提供自己的 API 密钥时才会激活：

服务 用途 传输的数据
Deepgram 实时语音识别 麦克风音频（实时流）
阿里云（通义千问） 实时语音识别 麦克风音频（实时流）

音频数据通过 WebSocket 实时传输，SOP Skill 不存储任何音频。请查阅这些提供商的隐私政策：

  • Deepgram 隐私政策
  • 阿里云隐私政策

4. 截图

SOP Skill 在录制会话期间捕获网页截图以记录您的工作流。这些截图：

  • 完全在您的浏览器中处理
  • 通过 Offscreen 文档中的 Canvas API 在本地标注
  • 永远不会上传到任何服务器
  • 仅嵌入在本地生成的 SOP 文档中

5. 权限说明

SOP Skill 仅请求核心功能所需的权限：

  • activeTab — 开始录制时访问当前标签页
  • sidePanel — 在 Chrome 侧栏中显示录制界面
  • storage — 在本地保存设置和 API 密钥
  • scripting — 将操作捕获脚本注入网页
  • clipboardWrite — 点击「复制」时将 SOP 文本复制到剪贴板
  • offscreen — 处理截图以标注点击位置
  • webNavigation — 录制期间检测页面导航

6. 儿童隐私

SOP Skill 不面向 13 岁以下儿童，也不会有意收集儿童的任何信息。

7. 政策变更

我们可能会不时更新本隐私政策。任何更改将在此页面上反映，并更新「最后更新」日期。

8. 联系方式

如果您对本隐私政策有疑问，请在我们的 GitHub 仓库中提交 Issue。
