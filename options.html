使用说明

选择一个语音提供商后，录制时即可实时转录语音为文字注释。

Deepgram 需要明确指定中文或英文语言；阿里云支持自动检测。

所有凭证均保存在浏览器本地存储中，不会上传到任何服务器。

如未配置 API Key 且未选择手动模式，点击开始录制时会提示错误。
